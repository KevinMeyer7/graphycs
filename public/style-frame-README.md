# Style Reference Frame

Place your style reference image here as `style-frame.png` (1280x720).

This image will be used as a visual anchor for all B-roll generation, ensuring style consistency across clips.

## Recommended Style
- Flat 2D comic aesthetic
- Soft gradients
- Teal/emerald accents
- Minimal palette
- Corporate training vibe

If no image is provided, the system will use a placeholder (generation will still work but with less style consistency).
